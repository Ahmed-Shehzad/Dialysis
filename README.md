# Dialysis PDMS

A learning platform for healthcare systems, dialysis domain workflows, and FHIR interoperability. Built with .NET 10, microservices, and HL7 v2 (IHE PCD).

## Quick Start

```bash
docker compose up -d
```

- **Gateway**: http://localhost:5001
- **Health**: http://localhost:5001/health
